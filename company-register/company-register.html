    POST users/company/admin/register/

    Company Admin creation/registration

    Parameter In Type Default Nullable Description

    email body string No User Email address
    password1 body string No Set Password
    password2 body string No Confirm Password
    role_name body string No User Role
    [AppSuperAdmin]
    user_type body string No Type of User
    [Application]
    username body string No User name
