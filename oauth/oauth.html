    Input parameters

    Parameter In Type Default Nullable Description
    client_id body string No Client ID from
    the OAuth provider
    client_secret body string No Client secret from
    the OAuth provider
    grant_type body None No Type of OAuth grant
    [Eg:- client_credentials,
    authorization_code]
