    HTTP Status Code

    2xx: Success

    Status Code Body Usecase
    200 OK { "message": "Success" } The web page was Successfully loaded (GET request)
    201 Created { "message": "xxxxxxxx content created"} Content was successfully created( POST request)
    204 No Content { "message": "No additional content"} Request was successful, but no additional content (response body is not necessary).

    3xx: Redirection

    Status Code Body Usecase
    302 Found { "message": "Resource found" } Requested resource resides temporarily under a different URL (temporary redirect).
    304 Not Modified { "message": "Not modified" } Resource hasn't been modified since the last request.

    4xx: Client Error

    Status Code Body Usecase
    400 Bad Request { "message": "Invalid request" } Server cannot process the request due to client error
    401 Unauthorized { "message": "Authentication required" } Requires user authentication
    403 Forbidden { "message": "Access denied" } Understood the request, but refuses to authorize it.
    404 Not Found { "message": "Resource not found" } Server can't find the requested resource
    405 Method Not Allowed { "message": "Method not allwoed" } Request is not allowed for the resource

    5xx: Server Errors

    Status Code Body Usecase
    500 Internal Server Error { "message": "An unexpected error occurred" } Server encountered an unexpected condition
    502 Bad Gateway { "message": "Invalid response from upstream server" } Received an invalid response from the upstream server.
    504 Gateway Timeout { "message": "Gateway timeout" } Did not receive a timely response from upstream server
